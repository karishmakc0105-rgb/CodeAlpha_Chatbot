# CodeAlpha Chatbot

This project is an AI-powered chatbot web application developed as part of the CodeAlpha Cloud Computing Internship.

## Features

* Instant chatbot responses
* Interactive web interface
* User-friendly design
* Cloud computing based project

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript

## Project Objective

The objective of this project is to create a chatbot capable of responding to user queries instantly through a web-based interface.

## How to Run the Project

1. Install Python
2. Install Flask using:

pip install flask

3. Run the application using:

python app.py

4. Open browser and visit:

http://127.0.0.1:5000
